# 3. Contains Duplicate

**LeetCode link:** https://leetcode.com/problems/contains-duplicate/

## Problem (paraphrased)

Return true if any value appears at least twice in the array; otherwise return false.

## Example

- Input: nums = [1,2,3,1]
- Output: true

## Constraints

1 <= nums.length <= 10^5; -10^9 <= nums[i] <= 10^9
