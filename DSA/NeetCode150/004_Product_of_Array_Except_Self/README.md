# 4. Product of Array Except Self

**LeetCode link:** https://leetcode.com/problems/product-of-array-except-self/

## Problem (paraphrased)

Return an array where each element is the product of all other elements except itself, without using division and in O(n) time.

## Example

- Input: nums = [1,2,3,4]
- Output: [24,12,8,6]

## Constraints

2 <= nums.length <= 10^5; -30 <= nums[i] <= 30
