# 2. Valid Anagram

**LeetCode link:** https://leetcode.com/problems/valid-anagram/

## Problem (paraphrased)

Given two strings, determine if one is an anagram of the other (same characters with same counts).

## Example

- Input: s = "anagram", t = "nagaram"
- Output: true

## Constraints

1 <= s.length, t.length <= 5 * 10^4; s and t consist of lowercase letters.
