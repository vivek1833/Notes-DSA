# 17. Minimum Window Substring

**LeetCode link:** https://leetcode.com/problems/minimum-window-substring/

## Problem (paraphrased)

Given strings s and t, return the smallest substring of s containing all chars of t (or empty if none).

## Example

- Input: s = "ADOBECODEBANC", t = "ABC"
- Output: "BANC"

## Constraints

1 <= s.length, t.length <= 10^5
