# 1. Two Sum

**LeetCode link:** https://leetcode.com/problems/two-sum/

## Problem (paraphrased)

Given an array of integers and a target, return indices of the two numbers that add up to the target. Exactly one solution exists and you may not use the same element twice.

## Example

- Input: nums = [2,7,11,15], target = 9
- Output: [0,1]

## Constraints

2 <= nums.length <= 10^4;
-10^9 <= nums[i] <= 10^9;
-10^9 <= target <= 10^9
