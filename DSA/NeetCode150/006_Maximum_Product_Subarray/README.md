# 6. Maximum Product Subarray

**LeetCode link:** https://leetcode.com/problems/maximum-product-subarray/

## Problem (paraphrased)

Find the contiguous subarray within an array (at least one number) which has the largest product.

## Example

- Input: nums = [2,3,-2,4]
- Output: 6

## Constraints

1 <= nums.length <= 2 * 10^4; -10 <= nums[i] <= 10
