# 19. Valid Parentheses

**LeetCode link:** https://leetcode.com/problems/valid-parentheses/

## Problem (paraphrased)

Given a string of brackets, determine if it's valid (properly closed and nested).

## Example

- Input: s = "()[]{}"
- Output: true

## Constraints

1 <= s.length <= 10^4; s consists of parentheses characters.
