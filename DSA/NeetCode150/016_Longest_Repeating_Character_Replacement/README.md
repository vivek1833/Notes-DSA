# 16. Longest Repeating Character Replacement

**LeetCode link:** https://leetcode.com/problems/longest-repeating-character-replacement/

## Problem (paraphrased)

Given a string and k, you may replace up to k characters to get the longest substring with all same letters; return its length.

## Example

- Input: s = "AABABBA", k = 1
- Output: 4

## Constraints

1 <= s.length <= 10^5; s consists of uppercase English letters.
