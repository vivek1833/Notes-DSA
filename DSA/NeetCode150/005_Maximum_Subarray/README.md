# 5. Maximum Subarray

**LeetCode link:** https://leetcode.com/problems/maximum-subarray/

## Problem (paraphrased)

Find the contiguous subarray with the largest sum and return that sum.

## Example

- Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
- Output: 6

## Constraints

1 <= nums.length <= 10^5; -10^4 <= nums[i] <= 10^4
