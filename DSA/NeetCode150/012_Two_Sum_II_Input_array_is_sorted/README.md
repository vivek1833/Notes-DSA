# 12. Two Sum II Input array is sorted

**LeetCode link:** https://leetcode.com/problems/two-sum-ii-input-array-is-sorted/

## Problem (paraphrased)

Given a sorted array, find 1-based indices of two numbers that add up to a target.

## Example

- Input: numbers = [2,7,11,15], target = 9
- Output: [1,2]

## Constraints

2 <= numbers.length <= 3 * 10^4; -1000 <= numbers[i] <= 1000
