# 9. 3Sum

**LeetCode link:** https://leetcode.com/problems/3sum/

## Problem (paraphrased)

Return all unique triplets [a,b,c] in the array that sum to zero.

## Example

- Input: nums = [-1,0,1,2,-1,-4]
- Output: [[-1,-1,2],[-1,0,1]]

## Constraints

0 <= nums.length <= 3000; -10^5 <= nums[i] <= 10^5
