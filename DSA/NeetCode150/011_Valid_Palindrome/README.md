# 11. Valid Palindrome

**LeetCode link:** https://leetcode.com/problems/valid-palindrome/

## Problem (paraphrased)

Check if a string is a palindrome considering only alphanumeric characters and ignoring cases.

## Example

- Input: s = "A man, a plan, a canal: Panama"
- Output: true

## Constraints

1 <= s.length <= 2 * 10^5
