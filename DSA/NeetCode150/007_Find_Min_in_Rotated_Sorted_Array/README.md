# 7. Find Min in Rotated Sorted Array

**LeetCode link:** https://leetcode.com/problems/find-minimum-in-rotated-sorted-array/

## Problem (paraphrased)

Array sorted and rotated; find the minimum element in O(log n) time.

## Example

- Input: nums = [3,4,5,1,2]
- Output: 1

## Constraints

1 <= nums.length <= 5000; -5000 <= nums[i] <= 5000
